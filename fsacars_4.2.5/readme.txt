================================================================================
Note! You are the installer!  Manual installation only! 
================================================================================

Create a new folder where you want to put fsacars (e.g., D:\FSACARS)

Extract the files from this zip to that folder.

You may need Run As Admin, disable antivirus or other programs.

After starting fsacars.exe, go to settings tab in acars and enter your credentials.

If you are setting this up for your server, you will need to edit the org.cfg
file, and use the server files (download separately from fsacars.com).

YOU DO NOT NEED A LICENSE TO USE THIS SOFTWARE FOR YOUR VIRTUAL AIRLINE,
just edit the server paths in org.cfg
